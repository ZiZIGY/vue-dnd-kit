# Introduction

Vue DnD Kit v2 is a lightweight drag & drop library for Vue 3. Built around a provider (`DnDProvider`) and composables: `makeDraggable`, `makeDroppable`, `makeSelectionArea`, `makeConstraintArea`.

## Key Features

- **Provider-based** — wrap your app with `DnDProvider`
- **Composables** — `makeDraggable`, `makeDroppable`, `makeSelectionArea`, `makeConstraintArea`
- **Keyboard support** — Enter/Space for start and drop, arrows for movement, Escape to cancel
- **TypeScript** — fully typed
- **No extra deps** — Vue as peer only
